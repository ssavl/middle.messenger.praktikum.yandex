
# SuperMessenger

#### Билдим и запускаем проект на локальном сервере
`npm run start`
#### Собираем и запускаем dev-сервер
`npm run dev`
#### Запускаем тесты
`npm run test`
#### Билдим
`npm run build`

Ссылка на PR https://github.com/ssavl/middle.messenger.praktikum.yandex/pull/5
